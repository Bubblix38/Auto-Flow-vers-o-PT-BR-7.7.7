# Auto Flow - Automação Para VEO AI 🚀

[Português](#português) | [English](#english)

---

## Português

**Auto Flow** é uma extensão poderosa para Google Chrome projetada para automatizar o fluxo de trabalho no **Flow AI (VEO)**. Com uma interface moderna, suporte total ao Português (Brasil) e diagnósticos avançados, ela transforma a maneira como você gera vídeos por IA.

### ✨ Funcionalidades

* **Automação de Fila**: Adicione múltiplos prompts e deixe a extensão processar tudo automaticamente.
* **Interface Premium**: Design moderno com abas coloridas, efeitos de brilho e organizado para máxima produtividade.
* **Suporte Bilingue**: Totalmente traduzido para Português (Brasil) e Inglês.
* **Diagnósticos em Tempo Real**: Sistema de logs detalhado com códigos de erro para identificar qualquer interrupção no site.
* **Modos Suportados**: Texto-para-Vídeo (Text-to-Video) e Imagem-para-Vídeo (Image-to-Video).
* **Configurações Persistentes**: Suas escolhas de modelo, proporção e quantidade de vídeos são salvas automaticamente.

### 🚀 Como Instalar

1. Faça o download deste repositório (Botão **Code** > **Download ZIP**).
2. Extraia o arquivo em uma pasta no seu computador.
3. Abra o Google Chrome e vá para `chrome://extensions/`.
4. Ative o **Modo do desenvolvedor** (canto superior direito).
5. Clique em **Carregar sem compactação** e selecione a pasta onde você extraiu a extensão.
6. Pronto! Fixe a extensão na sua barra de ferramentas e comece a criar.

## English

**Auto Flow** is a powerful Google Chrome extension designed to automate the workflow on **Flow AI (VEO)**. With a modern interface, full Portuguese (Brazil) support, and advanced diagnostics, it transforms the way you generate AI videos.

### ✨ Features

* **Queue Automation**: Add multiple prompts and let the extension process everything automatically.
* **Premium Interface**: Modern design with colored tabs, glow effects, and organized for maximum productivity.
* **Bilingual Support**: Fully translated into Portuguese (Brazil) and English.
* **Real-time Diagnostics**: Detailed log system with error codes to identify any interruptions on the site.
* **Supported Modes**: Text-to-Video and Image-to-Video.
* **Persistent Settings**: Your choices for model, aspect ratio, and video count are saved automatically.

### 🚀 How to Install

1. Download this repository (Click **Code** > **Download ZIP**).
2. Extract the file to a folder on your computer.
3. Open Google Chrome and go to `chrome://extensions/`.
4. Enable **Developer mode** (top right corner).
5. Click **Load unpacked** and select the folder where you extracted the extension.
6. You're done! Pin the extension to your toolbar and start creating.
